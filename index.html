I Miei Progetti

7 e mezzo

Sviluppo di una versione del classico gioco di carte "7 e mezzo", realizzata interamente in Java con un'interfaccia grafica creata tramite JavaFX. Il progetto si focalizza sull'applicazione di design pattern comportamentali (Strategy, Observer, State, Memento) per gestire con efficienza la logica di gioco, i ruoli dei giocatori e il salvataggio dello stato della partita. L'applicazione simula fedelmente le regole tradizionali e include un giocatore gestito dal computer.

GameGuessr

Un gioco a quiz multiplayer basato su un'architettura di rete P2P ibrida e decentralizzata. Sviluppato in Python con interfaccia grafica in Pygame, il progetto utilizza un server centrale per coordinare la connessione iniziale dei peer. Successivamente, un peer viene eletto "presentatore" e gestisce la partita comunicando direttamente con gli altri giocatori, inviando domande e validando le risposte in tempo reale.

ParthenoAnswer

Progettazione e sviluppo di "ParthenoAnswer", una web app stile forum universitario creata per gli studenti dell'Università Parthenope. L'applicazione è stata realizzata con Flask (Python), JavaScript e MongoDB come database NoSQL per il backend e html, css e bootstrap per il front end, la sua particolarità è l'integrazione con le API ufficiali dell'ateneo per automatizzare il login e recuperare dati sulla carriera.
